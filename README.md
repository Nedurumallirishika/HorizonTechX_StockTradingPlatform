# 📈 HorizonTechX Stock Trading Platform

A Java console-based application developed as part of the **Horizon TechX Java Programming Internship**. This project simulates a basic stock trading platform where users can view stocks, buy and sell shares, manage a portfolio, and track their wallet balance.

---

## 📌 Features

- 📊 View Available Stocks
- 💰 Buy Stocks
- 💸 Sell Stocks
- 📁 View Portfolio
- 👛 Check Wallet Balance
- ✅ Menu-Driven Console Application
- 🔄 Updates Stock Quantity After Transactions

---
